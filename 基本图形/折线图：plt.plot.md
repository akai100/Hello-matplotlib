+ color

  + 颜色英文词：'red'/'blue'/'green'/'black'/'gray'
 
  + 十六进制颜色：'#FF0000'（红）、'#0000FF'（蓝）等；
 
  + RGB、RGBA 元组：(1,0,0)（红）、(0,1,0,0.5)（半透明绿）；
 
  + 单字母缩写：'r'（红）、'b'（蓝）、'g'（绿）、'k'（黑）、'w'（白）等

+ linestyle(简写 ls，指定折线的线型（实线 / 虚线等）)

  '-'（实线，默认）、'--'（双虚线）、'-.'（点划线）、':'（点虚线）、'None'/''（无线条，仅显示标记）

+ linewidth(简写 lw，指定折线的宽度（像素值）)

  数值类型：1、2.5、5 等（默认值为 1）

+ alpha: 指定折线的透明度（全局透明度，含标记）

  浮点型：0（完全透明）~ 1（完全不透明），默认 1

+ marker：指定数据点的标记形状

  '.'（点标记）、'o'（圆形标记）、's'（正方形标记）、'^'（上三角形标记）、'*'（星形标记）、'x'（叉形标记）、'+'（十字标记）等
